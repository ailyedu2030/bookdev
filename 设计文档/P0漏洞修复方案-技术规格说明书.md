## P0-1: H-001 引用幻觉自我强化循环

### 问题描述
错误引用被多次引用后"固化"，形成自我强化的错误闭环。

### 技术架构

```
┌─────────────────────────────────────────────────────────────────┐
│                   事实版本戳系统（Fact Timestamping）              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐     │
│  │  事实注册表   │    │  引用验证器   │    │  传播追踪器   │     │
│  │  FactTable   │◄───│   Verifier   │◄───│  Propagator  │     │
│  └──────────────┘    └──────────────┘    └──────────────┘     │
│         │                   │                   │              │
│         ▼                   ▼                   ▼              │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │              不可变日志（Immutable Log）                 │   │
│  │         SHA-256(content) + version + timestamp          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 核心数据结构

### 问题描述

```
│  数据血缘图:                                                    │
│                                                                 │
│    [原始数据] ──派生1──> [数据A] ──派生2──> [数据B]            │
│        │                                    │                   │
│        │                                    └──派生3──> [数据C]  │
│        │                                                         │
│    fact_hash                        propagation_chain: [A→B→C] │
│                                                                 │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │              数值核实引擎（Tier1核心）                    │   │
```

## P0-3: H-004 法规条文幻觉合成

### 问题描述
AI捏造不存在的法规条款，如《人工智能法》第28条。

### 技术架构

```

│                                                                 │
│  引用格式:                                                       │
│  [@reg:{law_name}:{article_num}:{content_hash}]               │
│                                                                 │
│  示例:                                                          │
│  [@reg:人工智能法:28:a3f5b8...]                                │
│                                                                 │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │              三级核实引擎                                  │   │
│  │                                                           │   │
│  │  Tier 1: 法规存在性 ──→ 国家法规数据库API                │   │
│  │       ↓                                                   │   │
│  │  Tier 2: 条款存在性 ──→ 条款编号范围校验                 │   │
│  │       ↓                                                   │   │

│  └──────────────────────────────────────────────────────────┘   │

│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```python
        """引用记录（不可篡改）"""

            self,

            original_source: str,    # 原始来源URL

            doi: str | None,         # DOI（如有）
            created_at: datetime,
            created_by: str

            self.title = title
            self.doi = doi
            self.created_at = created_at
            self.created_by = created_by
            self.is_locked = True  # 锁定后不可修改
```
